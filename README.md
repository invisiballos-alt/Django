# Educational Django & JavaScript Web Application

Пет-проект, созданный в рамках изучения архитектуры веб-фреймворка Django и интеграции интерактивной клиентской логики (JavaScript) с бэкендом.

## Стек
* **Backend:** Python 3, Django (MVT architecture)
* **Frontend:** JavaScript (Vanilla JS), HTML5, CSS3

## Реализованный функционал и изученные концепции
* **Архитектура Django:** Настройка структуры проекта, конфигурирование маршрутизации (`urls.py`) и создание базовых представлений (`views.py`).
* **Шаблонизация:** Практическое применение встроенного движка шаблонов Django (DTL) — использование наследования (`{% extends %}`) и изолированных блоков (`{% block %}`) для модульной структуры страниц.
* **Интеграция фронтенда:** Реализация динамической логики на стороне клиента (обработка игровых событий, обновление состояния баланса и результатов без перезагрузки страницы).
* **Управление статикой:** Настройка и подключение статических файлов (CSS, JS) в экосистеме Django.
